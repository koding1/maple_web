## 소개
Maple_web은 온라인 게임 '메이플스토리' 에 대한 데이터를 기반으로,     
크게 게임에 존재하는 직업에 대한 인기도를 측정하여 제공하는 서비스와     
유저 성향 분석으로 직업을 추천해주는 서비스로 나눌 수 있습니다.
